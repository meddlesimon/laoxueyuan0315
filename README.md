
# 老孙教育科技 - 自动化学生专属课程推荐系统

## 📖 项目简介

**自动化学生专属课程推荐系统** 是专为【老孙教育科技】定制开发的 Web 应用程序。该系统旨在将繁杂的学生问卷/测评数据（CSV格式）转化为排版精美、高度个性化的 **PDF 课程推荐方案**。

系统内置了智能规则引擎，能够根据学生的年级（覆盖学前至高中）、成绩、薄弱项以及使用的学习机品牌（学而思/科大讯飞），自动匹配最合适的课程路径，极大地提高了教务人员的工作效率和方案的专业度。

## ✨ 核心功能

*   **📊 智能数据解析**：
    *   支持 CSV 文件批量导入。
    *   自动识别并清洗数据，兼容多种问卷格式表头。
    *   智能判定学生类型：**K12（小初高）** 与 **学前启蒙** 分流处理。
    *   自动归一化年级格式（如：将 "High 1", "10年级" 统一为 "高一"）。
*   **🧠 自动化课程匹配**：
    *   基于**分数段**和**排名**自动计算学科等级（L1-L4）。
    *   根据**薄弱项**（如：几何、文言文、拼音）优先推荐相关模块。
    *   区分 **学而思** 与 **科大讯飞** 两套不同的课程体系数据库。
    *   针对学前阶段提供专门的启蒙（识字、思维、RAZ阅读等）逻辑。
*   **📝 可视化编辑与诊断**：
    *   提供以“孙老师”口吻生成的个性化诊断语，包含学情分析、习惯建议与行动公约。
    *   支持手动编辑学生档案、修改推荐课程、添加/删除课程模块。
    *   实时预览修改效果。
*   **💾 高性能数据存储**：
    *   采用 **IndexedDB** 替代 LocalStorage，支持存储海量学生档案，解决了浏览器存储配额限制问题。
    *   数据仅保存在本地浏览器，保障隐私安全。
*   **📄 专业级 PDF 导出**：
    *   **A4 标准排版**：封面、诊断页、学科详情页自动分页。
    *   **批量导出**：支持多选学生，一键生成 ZIP 压缩包下载。
    *   **视觉设计**：内置精美的背景、图标与配色体系，区分不同学科风格。
*   **🔎 高效管理**：
    *   支持按姓名、年级搜索。
    *   支持按**问卷提交日期**进行筛选（时间段筛选/特定日期点选）。

## 🛠 技术栈

*   **前端框架**: [React 19](https://react.dev/)
*   **语言**: [TypeScript](https://www.typescriptlang.org/)
*   **样式库**: [Tailwind CSS](https://tailwindcss.com/)
*   **构建工具**: [Vite](https://vitejs.dev/)

*   **本地存储**: [idb](https://github.com/jakearchibald/idb) (IndexedDB Wrapper)
*   **图标库**: [Lucide React](https://lucide.dev/)
*   **工具库**: JSZip, FileSaver

## 🚀 快速开始与部署

### 本地开发

1.  **安装依赖**:
    ```bash
    npm install
    ```
2.  **启动开发服务器**:
    ```bash
    npm run dev
    ```

### ☁️ 部署到腾讯云 EdgeOne (Pages)

1.  **构建生产版本**:
    在本地项目根目录运行以下命令，生成静态文件：
    ```bash
    npm run build
    ```
    构建完成后，会生成一个名为 `dist` 的文件夹。

2.  **上传至 EdgeOne**:
    *   登录 [腾讯云 EdgeOne 控制台](https://console.cloud.tencent.com/edgeone)。
    *   进入 **Pages (静态网站托管)** 服务。
    *   创建一个新项目或选择现有项目。
    *   选择 **“手动上传”** 方式。
    *   将本地生成的 `dist` 文件夹上传。
    *   部署完成后，您将获得一个访问域名，即可在线使用系统。

## 📂 CSV 数据格式规范

导入的 CSV 文件应包含以下核心字段（支持标准字段名或问卷常见问题描述）：

| 标准字段名 | 问卷常见表头示例 | 说明 |
| :--- | :--- | :--- |
| **student_name** | 请输入孩子的姓名 / 学生姓名 | 必填 |
| **grade** | 您的孩子现在是几年级 / 年级 | 必填，支持中文年级或数字 |
| **math_score** | 在校数学成绩 | K12 必填，支持分数或区间 |
| **chinese_score** | 在校语文成绩 | K12 必填 |
| **english_score** | 在校英语成绩 | K12 必填 |
| **weak_points** | 最想提升的模块 / 孩子现阶段主要想提高的科目 | 用于精准匹配课程 |
| **study_duration** | 愿意投入的时间 / 课外学习时长 | 用于诊断语生成 |
| **machine_brand** | 学习机品牌 / 设备 | 用于区分课程库 (默认学而思) |
| **submit_time** | 提交时间 / 答题结束时间 | 用于按日期筛选 |

> **注意**: 系统会自动尝试匹配包含关键词的列头。如果是学前儿童，系统会根据年级（小/中/大班）或特定的兴趣字段自动切换至学前模式。

## ⚠️ 注意事项

*   **数据安全**: 所有数据均存储在您浏览器的 IndexedDB 中。**清除浏览器缓存会导致数据丢失**，请定期备份原始 CSV 文件。
*   **PDF 生成**: 批量生成 PDF 时需要消耗一定的浏览器内存，建议一次批量导出不要超过 50-100 人，以免浏览器崩溃。测试一下
